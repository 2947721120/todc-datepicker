Google Datepicker

Google Datepicker is a fork of Andrew Rowls' bootstrap-datepicker and provides a themed datepicker component which fits naturally with Google Bootstrap.

Date Formatting

            
The desired formatting for the selected date can be set via javascript during instantiation,
              or in the input element's markup.

            

                

                  

                    
                    
                  

                

            

        

        

            
Example Code

            

            
<input id="cal3" type="text" data-date-format="mm-dd-yyyy"/>
